# Movie_Rating_Analysis

IMDb Movie Performance Analytics

A comprehensive data analysis project examining the relationships between movie metrics, including budget, runtime, and audience ratings.

## 📊 Project Overview
This project goes beyond basic descriptive statistics to perform a deep-dive analysis of movie performance. By leveraging the IMDb dataset, we investigate how various production factors influence critical reception and commercial success.

## 🛠 Key Analytical Techniques
*   **Data Wrangling:** Cleaned raw datasets and performed feature engineering to create temporal categories (e.g., Decades).
*   **Outlier Detection:** Utilized the Interquartile Range (IQR) method to identify and isolate extreme blockbusters that skew statistical averages.
*   **Temporal Analysis:** Analyzed long-term trends in movie ratings across decades to detect shifting audience preferences.
*   **Multivariate Visualization:** Employed logarithmic scaling and multi-dimensional scatterplots (hue, size, and axis mapping) to reveal complex correlations between Votes, Ratings, and Genre.

## 📈 Key Insights
*   **Revenue vs. Rating:** Investigated the correlation between blockbuster status and audience perception.
*   **Genre Longevity:** Identified which genres maintain consistent rating performance over time.
*   **Temporal Bias:** Visualized how audience standards for "good" movies have evolved from the 1970s to the present.

## 🧰 Tech Stack
*   **Languages:** Python
*   **Libraries:** `pandas` (Aggregation & Wrangling), `seaborn` & `matplotlib` (Statistical Visualization), `numpy` (Numerical Processing).
